# 開發即時遠端遙控機械手臂之六自由度操縱裝置
隨著人機互動技術的演進，遠端遙控機器人技術已成為人機協作領域的關鍵技術。傳統的鍵盤、滑鼠和搖桿等輸入設備僅適用於離散式指令輸入，難以滿足對機器人進行連續、多自由度運動控制的複雜任務需求，因此為了實現更精細且連續的運動控制，開發各種具備高自由度的操縱桿裝置已成為研究重點。

有鑑於現行主控設備普遍面臨自由度不足、工作空間受限及成本高昂等技術瓶頸，本研究所提出的設計，其核心目標在於提供一個兼具高自由度、大工作空間與成本效益的解決方案，藉此提升主從控制系統的性能與實用性。本研究提出設計新穎的六自由度操縱桿裝置，並進行裝置設計、運動學的推導及工作空間的計算。最後整合人機介面、操縱桿裝置和機械手臂，建構完整的主從式控制系統。

在實驗方面，本研究進行空間位置驗證實驗，以驗證裝置精度。由於裝置製造公差和組裝過程誤差，造成運動學模型中的幾何參數與實體機構產生偏差，因此本研究基於類神經網路，建立誤差補償模型進行位置誤差修正。最後經誤差補償後的輸出結果，X、Y、Z的位置平均絕對值誤差分別為0.221 cm、0.210 cm、0.226 cm，位置均方根誤差分別為0.263 cm、0.263 cm、0.280 cm。誤差補償後的Roll、Pitch、Yaw的角度平均絕對值誤差分別為2.250˚、1.890˚、2.981˚，角度均方根誤差分別為2.754˚、2.316˚、3.503˚ 。最後透過實際操作之抓取與放置實驗，驗證系統的可行性，各物體平均放置率為70%，平均花費時間約為37.3 s，平均花費時間標準差為2.8 s，實驗結果驗證了本系統所達成的定位精度，以及主從式控制系統的可行性。未來可整合視覺回饋與力回饋，增加操作時的臨場感，以及優化與機器人之間的通訊系統，實現長距離的遠端操控。
